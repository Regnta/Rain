## 描述
找遍了一些主题站上现有的皮肤，没有一款符合自己的心意，并且样式看的头昏眼花不知道从何改起，为了解决这个问题就从头借助AI来 `Vibe Coding` 了一款主题，由AI来快速设计一些组件的样式自己加加减减修改到目前这个状态。

> 🎉 主题处于持续开发的状态，代码及许多功能不算完善。

## 功能列表

- [x] Markdown 基本渲染
- [x] Katex 数学公式
- [x] Mermaid 流程图、甘特图、序列图
- [x] Echarts 饼图、柱状图
